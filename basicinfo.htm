Настоящий документ содержит необходимые сведения для работы с программой dicom2fmm.exe,

предназначенной для использования в лечебном процессе на Комплексе протонной терапии «ПРОМЕТЕУС».

Программа dicom2fmm.exe позволяет конвертировать файлы полученные на КТ томографах в международном формате DICOM, в файлы во внутреннем формате производителя (FMM), являющимсявходными для программы «Система планирования протонной лучевой терапии».

 

Данное руководство содержит:

•технические и программные требования, необходимые для выполнения программы.

•инструкции по установке и конфигурированию программы.

•последовательность действий, необходимых для запуска,
выполнения и завершения программы.

•основные сведения необходимые пользователю для понимания,
как и зачем работает программа.

•описание интерфейса пользователя.

•порядок действий и этапы выполнения программы.

•описание результата работы программы.

•тексты сообщений, выдаваемые в ходе выполнения программы.
