Запуск программы из командной строки

Запуск программы из командной строки позволяет задавать особые параметры текущего сеанса работы программы, изменить настройки и получить справочную информацию о параметрах командной строки.

Синтаксис команды запуска:

[<путь_к_программе>]dicom2fmm.exe [параметры командной строки]

 

dicom2fmm.exe,  параметры командной строки:

Всегда заключайте в кавычки все пути и имена файлов, содержащие пробелы.

-help

Отображается справочная информация о параметрах командной строки, программа завершает работу.

 

-id

ID пациента в базе данных

-id 443

 

-inputdir

Имя входной директории содержащей данные пациента

-inputdir

P:\P_DATA\ЯковлеваЕлизаветаЛеонардовна_443\Studies

 

-outputdir

Имя выходной директории, в которую будет записан выходной FMM файл

-outputdir

P:\P_DATA\ЯковлеваЕлизаветаЛеонардовна_443\tomo

 

-basedir

Необязательный параметр. Указывает на коренную директорию, где хранятся данные всех пациентов.

-basedir P:\P_DATA

-itomo

ID входной томограммы в базе данных

-itomo 1307

-pre

ID предписания в базе данных

-pre 442

-type

Тип входной томограммы

0 - DAT томограмма получена со встроенного томографа
        + DICOM томограмма с контурами

 

1 - сторонняя DICOM томограмма с контурами

Пример командной строки:

 

dicom2fmm.exe -id 443 -itomo 1307 -pre 442 -fract 1 -dose 21.8

           -inputdir P:\P_DATA\ЯковлеваЕлизаветаЛеонардовна_443\Studies

           -outputdir P:\P_DATA\ЯковлеваЕлизаветаЛеонардовна_443\tomo -type 0
