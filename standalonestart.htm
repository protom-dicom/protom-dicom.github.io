Запуск программы с помощью ярлыков

Ярлык на рабочем столе для запуска dicom2fmm.exe в автономном режиме.

 

Чтобы запустить программу с помощью ярлыка на рабочем столе, дважды кликните мышью по ярлыку.

После запуска появится стартовая страница, которая служит для выбора директории с оконтуренной томограммой в формате DICOM-RT.

 

Стартовое окно программы

  DICOM директорию можно выбрать несколькими способами:

 

•выбор пациента и директории из базы  данных

•ввод директории с помощью командной строки.

•с помощью обозревателя директорий.

 
  Последние два способа используются для просмотра DICOM томограмм, тестирования программы,

но без возможности записи результата в базу данных.
